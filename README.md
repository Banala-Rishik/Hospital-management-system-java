🏥 Hospital Management System (HMS)
Java | JDBC | MySQL


📌 Overview

Developed a console-based Hospital Management System to manage patient records, doctor details, and appointment scheduling. The application demonstrates backend development concepts by integrating Java with a MySQL database using JDBC.

🚀 Key Features

Implemented CRUD operations for managing patient records.

Designed a doctor management module to store and retrieve specialization details.

Developed an appointment scheduling system with logic to avoid double-booking of doctors.

Used PreparedStatement for secure and efficient database interactions.

Maintained data integrity using primary and foreign key constraints.

🏗️ Tech Stack

Programming Language: Java

Database: MySQL

Connectivity: JDBC

Concepts Used: OOP, SQL, Exception Handling

📊 Database Design

Patients: Stores patient ID, name, age, and gender.

Doctors: Stores doctor ID, name, and specialization.

Appointments: Links patients and doctors with appointment date using foreign keys.

⚙️ Responsibilities / Implementation

Designed and implemented database schema using SQL.

Established database connectivity using JDBC.

Wrote modular Java code for handling different system components.

Implemented validation and basic scheduling logic.
